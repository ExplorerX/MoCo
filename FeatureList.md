## 2. V2 产品原则

1. **声音优先**：优先建立声音与字符之间的直接反射，视觉点划只作为解释和辅助。
2. **训练方式与训练内容分离**：识别、听抄、发报是训练方式；字符、词组、长文、Q 简语等是可复用内容包。
3. **一套引擎，多种预设**：不同训练入口共用会话、音频、输入、评分、持久化和统计能力，不复制业务逻辑。
4. **科学渐进**：采用 Koch 渐进学习、Farnsworth 间隔、错题强化和可解释的薄弱项推荐。
5. **专业但不劝退**：入门用户默认只看到下一步；速度、信道模拟和专业通信内容按需展开。
6. **低延迟反馈**：声音、触摸、键盘和视觉反馈必须及时、稳定且可校准。
7. **本地与隐私优先**：核心课程和工具离线可用；自由输入与自定义正文默认不写入训练日志。
8. **跨端一致**：训练规则、内容标识、评分和数据结构跨平台一致，平台差异仅限系统能力。

### 3.3 统一训练会话

所有计分训练使用同一条主流程：

1. 选择功能域和训练预设。
2. 选择或沿用内容包。
3. 使用快速开始，或打开统一训练设置。
4. 进入专注会话并逐题或连续作答。
5. 即时保存结果，生成反馈和能力数据。
6. 从结果页重练错题、提高难度或切换训练方式。

“听抄训练台”是听抄预设的统一配置与会话外壳，不作为与字母听抄、词组听抄平级的内容项目。

## 4. 专业术语基线

- 使用 **Morse Code / 摩斯电码** 作为产品通用名称；无线电语境可使用 **CW**。
- 使用 **Q 简语** 或 **Q 信号（Q Signals）**，不使用“QC 简语”。
- `CQ` 表示广泛呼叫，与 Q 简语分开归类。
- `QTH`、`QRM`、`QRN`、`QRS`、`QSY` 等归入 Q 简语内容包。
- `CQ`、`DE`、`TNX`、`73` 等归入 CW 常用缩略语内容包。
- `AR`、`SK`、`BT`、`AS` 等归入程序信号（Prosigns）；连写发送规则必须由内容数据明确标记。
- 使用 **数字简码（Cut Numbers）**，并为映射标记使用场景和方案；不得暗示所有比赛采用完全相同映射。
- **中文电码** 指汉字与四位数字代码之间的映射。汉字转 Morse 的完整链路为“汉字 → 中文电码数字组 → 数字 Morse”，不得描述为国际 Morse 直接编码汉字。

### 13.1 性能与可靠性

- 输入反馈优先于非关键动画、转换预览和统计计算。
- 连续长文播放不得出现随长度增加的累计时序漂移。
- 低性能移动设备上仍应保持按键、声音和实时视觉反馈同步。
- 训练单题结果及时持久化；V2 数据与旧版数据库物理隔离。
- 音频或平台能力不可用时，页面明确降级，不伪造训练结果。

### 13.2 安全、隐私与内容来源

- 默认仅在本地保存训练数据。
- 自由拍发与转换正文默认不写入训练日志或云端分析。
- 专业内容包记录来源、版本、许可和内部数据版本。
- 中文电码必须使用可追溯且许可允许的码表；未确认数据源前不得发布转换结果。
- 日志不得包含密码、令牌、完整输入正文或其他敏感数据。

### 13.3 可测试性与平台兼容性

- 编解码、时序、内容生成、评分和复习算法可脱离界面测试。
- V2 训练预设与内容包使用稳定 ID；稳定性承诺从 V2 新会话开始。
- Web 端优先支持当前主流 Chromium、Safari 和 Firefox。
- 每次发布至少验证一种触摸环境和一种桌面物理键盘环境。

## 14. V2 结构重构验收标准

- [ ] 首页之外的一级功能入口与四大功能域一一对应。
- [x] 旧四种练习能力均可从新的功能域入口开始；V2 结果和历史从空数据开始。
- [ ] 相同内容包不在多个功能域维护重复数据。
- [ ] 听抄训练台是统一配置壳，字母、词组等以预设进入。
- [ ] 自由发报迁移为“自由拍发”，原输入、声音和时长反馈无回归。
- [ ] Q 简语、CQ、CW 缩略语、程序信号和数字简码术语不混用。
- [ ] 未实现的内容不显示可点击空页面或虚假结果。
- [x] 旧设置、会话、课程进度、错题和统计不读取；旧 URL 直接进入未找到页。
- [ ] 移动端一级导航不超过五项，专注会话继续隐藏全局导航。
- [ ] FeatureList、ProductSpec 和 InformationArchitecture 的名称、优先级和路由一致。

## 15. 当前不进入本次结构重构的范围

- 完整 QSO 仿真和复杂信道模拟。
- 系统级输入法。
- 云同步、社区、排行榜、付费和广告系统。
- 外接电键的完整硬件支持。
- 未确认码表来源和许可前的中文电码生产数据。
- 一次性实现全部 P1/P2 内容；本次先重构结构并迁移现有能力。
